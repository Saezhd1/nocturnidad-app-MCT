# MCT Nocturnidad - Web para cálculo de complemento nocturno

Aplicación web (Flask) para subir varios PDFs con plantillas de registro de jornada, extraer la primera tabla (columnas **Fecha**, **HI**, **HF**), calcular minutos nocturnos según reglas indicadas y generar un informe PDF descargable con resumen diario, mensual, anual y global.

## Requisitos
- Python 3.10+ recomendado
- Dependencias listadas en `requirements.txt`
